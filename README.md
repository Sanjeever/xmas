# xmas

Show a snowman on browser.
Build with Vue3 + Vite + Typescript

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```
